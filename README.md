# Statistics Service

Сервис для сбора и анализа данных с устройств. Реализован в соответствии с техническим заданием.

## Технологии

- **FastAPI** — REST API
- **PostgreSQL** — хранение данных
- **Celery + Redis** — асинхронная аналитика
- **Docker + Docker Compose** — контейнеризация
- **Locust** — нагрузочное тестирование

## Функциональные требования

| Требование | Статус |
|------------|--------|
| Сбор статистики с устройства (x, y, z) | ✅ |
| Анализ статистики за период и всё время | ✅ |
| Результаты: min, max, count, sum, median | ✅ |
| Поддержка пользователей устройств* | ✅ |
| Аналитика по пользователю (общая + по устройствам)* | ✅ |

## API Эндпоинты

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/users` | Создать пользователя |
| POST | `/devices` | Создать устройство |
| POST | `/devices/{id}/data` | Добавить показания |
| POST | `/analytics/device/{id}` | Статистика устройства |
| POST | `/analytics/user/{id}` | Статистика пользователя |
| GET | `/analytics/result/{task_id}` | Получить результат |

## Запуск сервиса

```bash
docker compose up --build
```

Сервис доступен: http://localhost:8000

Документация API: http://localhost:8000/docs

Locust: http://127.0.0.1:8089
